A) Project Structure - top

Top-level folders

src Source folder
public Static assets to be served
scripts A custom script for extracting Ant Design stylesheet
template Including code template for a page & its sub-component

Src-level folders

app App router
components Common components for the app
lib Including common functions, hooks, utils
services Backend services to get data from API
state App State
theme Theme configuration

App-level folders

_components Shared components
(crm) Including all pages for the admin panel. The folder name is fere to change if you needed. Changing of this folder name will not affect the current routing.
api API Routes
auth Authentication module
pages Static pages

B) Routing - top

There are 2 primary layouts which nested under the root layout.

The application was organized following by Next.js guideline which can be found at https://nextjs.org/docs/app/building-your-application/routing

Root layout /src/app/layout.js
App layout /src/app/(crm)/layout.js
Auth layout /src/app/auth/layout.js

Your new features should be placed into src/app/(crm) folder. Each folder in a route represents a route segment. Each route segment is mapped to a corresponding segment in a URL path.

Source from: next.js

C) Theme Setting - top

If you would like to make a custom theme such as color, font size, border of web components, you can find all theme settings under /src/theme/themeConfig.js

Here is default configuraton for the Adkit Admin Template

const theme = {
	token: {
		fontSize: 14,
		colorPrimary: '#00B96B',
		borderRadius: 3,
	},
	components: {
		Button: {
		algorithm: true,
		onlyIconSize: 14,
		},
	},
};
		
export default theme;

All the theme configuration can be found on Ant Design website. https://ant.design/docs/react/customize-theme
